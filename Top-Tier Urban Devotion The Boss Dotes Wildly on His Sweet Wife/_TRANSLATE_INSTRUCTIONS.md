# Translation instructions (shared by all chapter-translation agents)

You translate ONE chapter of the Traditional-Chinese web novel
《都市頂級偏愛：大佬嬌妻肆意寵》 (English title: "Top-Tier Urban Devotion: The Boss
Dotes Wildly on His Sweet Wife") into fluent, publication-quality English.

## Source & output
- Source: `都市頂級偏愛：大佬嬌妻肆意寵/chapters/NNN.txt` — 3 header lines
  (Chinese title, source URL, blank), then the chapter body.
- Output: `都市頂級偏愛：大佬嬌妻肆意寵/english/NNN.txt`, written with the Write tool.
- Output format, exactly:
  - Line 1: `Chapter <N> — <English chapter title>`
  - Line 2: the source URL (copy from source line 2)
  - Line 3: blank
  - Then the English translation, one line per source paragraph.

## Style
- Natural, idiomatic English a native reader would enjoy — NOT literal/machine
  translation. Modern romance / revenge web-novel tone. Keep dialogue snappy.
- Preserve paragraph structure and every line of dialogue. Do NOT summarize or omit anything.
- Render quoted design/work names in English quotes (e.g. "Fleeting Life" 浮生,
  "Cracks and Rebirth" 裂痕與新生, "Nirvana" 涅槃, "Listening to the Sea" 聽海,
  "Daybreak" 破曉, "Beginning" 初).
- Romanize any names not in the glossary using pinyin without tone marks, consistently.
